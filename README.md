# ゆで卵タイマー

鍋で作るゆで卵用のスマホ向け2段階タイマーです。

## 公開URL

https://codsworks.github.io/yude-tamago-timer/

## 使い方

1. 鍋に1cmの水を入れる
2. 冷蔵庫から卵をだして鍋にいれる
3. 鍋を火にかける
4. 沸騰したら中火にして蓋をする
5. スタートをタップ
6. アラームが鳴ったら火を消す
7. もう一度アラームが鳴ったら完成

## 時間設定

1回目のアラームはスタートから4分後です。

2回目のアラームは以下から選べます。

- 半熟: 3分
- すこし半熟: 4分
- 完熟: 5分

## デバッグモード

短い時間で動作確認できます。

https://codsworks.github.io/yude-tamago-timer/?debug

デバッグ時:

- 1回目のアラーム: 8秒
- 2回目のアラーム: 6秒 / 8秒 / 10秒

## ホーム画面に追加

スマホで公開URLを開き、ブラウザの共有メニューから「ホーム画面に追加」できます。

アイコン画像とWeb App Manifestを同梱しています。
