# ゆで卵タイマー

鍋に1cmの水だけで作る、ゆで卵用のスマホタイマーです。

沸騰したらスタート。  
1回目のアラームで火を消し、2回目のアラームで完成します。

半熟・すこし半熟・完熟を選べるので、いつものゆで卵を作りやすくします。

## 公開URL

https://codsworks.github.io/yude-tamago-timer/

スマホで開いて、ホーム画面に追加して使うのがおすすめです。

## 特徴

- スマホ向けのシンプルな1画面タイマー
- 1回目のアラームで火を消す2段階方式
- 半熟 / すこし半熟 / 完熟を選択
- 好きなアラーム音を選択
- タイマー中の画面消灯を抑制
- ホーム画面追加用のアイコンつき

## 使い方

1. 鍋に1cmの水を入れる
2. 冷蔵庫から卵をだして鍋にいれる
3. 鍋を火にかける
4. 沸騰したら中火にして蓋をする
5. スタートをタップ
6. アラームが鳴ったら火を消す
7. もう一度アラームが鳴ったら完成

## 時間設定

1回目のアラームは、スタートから4分後です。

2回目のアラームは、火を消してからの蒸らし時間です。

- 半熟: 3分
- すこし半熟: 4分
- 完熟: 5分

## デバッグモード

短い時間で動作確認できます。

https://codsworks.github.io/yude-tamago-timer/?debug

デバッグ時:

- 1回目のアラーム: 8秒
- 2回目のアラーム: 6秒 / 8秒 / 10秒

## ホーム画面に追加

スマホで公開URLを開き、ブラウザの共有メニューから「ホーム画面に追加」できます。

iPhoneの場合:

1. Safariで公開URLを開く
2. 共有ボタンをタップ
3. 「ホーム画面に追加」を選ぶ

Androidの場合:

1. Chromeで公開URLを開く
2. メニューを開く
3. 「ホーム画面に追加」を選ぶ
